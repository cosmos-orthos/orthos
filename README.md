# orthos

High-performance linear algebra library written in Rust with Python bindings.

## Installation

**Python:**
```bash
pip install orthos
```

**Rust:**
```toml
[dependencies]
orthos = "0.1"
```

- [Python documentation](https://pypi.org/project/orthos/)
- [Rust documentation](https://docs.rs/orthos)
